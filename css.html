CSS — Presentation and Layout

CSS (Cascading Style Sheets) controls the visual design and layout of web pages. While HTML is responsible for structure, CSS defines how content looks — colors, fonts, spacing, positioning, and responsiveness. Three main concepts shape CSS behavior:

  • Cascade: When multiple rules apply, the most specific or latest one takes effect.
  • Specificity: Determines which selector has higher priority (id > class > type).
  • Inheritance: Some properties (like font-family and color) naturally pass from parent to child elements.

Modern web design uses Flexbox and CSS Grid for powerful layouts. Responsive design adapts pages to any screen size using media queries and relative units.

Selectors & Specificity
/* Type, class, id */
p { line-height: 1.6; }
.note { color: #333; }
#highlight { background: #000; color:#fff; }

/* Attribute & pseudo-classes */
input[type="email"]:focus { outline: 2px solid #000; }

/* Specificity tips: prefer classes over ids for reusable styling */

Selectors target elements. Type selectors apply to all elements of a kind, classes allow reusable styles, and IDs are unique. Attribute selectors and pseudo-classes (like :hover, :focus) allow more advanced styling. Good practice: use classes for most styling to keep code modular and maintainable.

Flexbox & Grid
/* Flexbox */
.container { display:flex; gap:1rem; flex-wrap:wrap; }
.item { flex: 1 1 240px; }

/* Grid */
.grid {
  display:grid;
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
  gap: 1rem;
}

Flexbox is best for one-dimensional layouts (arranging items in a row or column). Grid excels at two-dimensional layouts, letting you define rows and columns explicitly. Together, they replace older, less flexible techniques like float-based layouts.

Responsive Techniques

@media (max-width: 768px) {
  .sidebar { display:none; }
  .hero { padding: 1rem; }
}

Media queries adjust layouts based on device width, making designs mobile-friendly. For example, sidebars can be hidden on smaller screens, while spacing can be reduced for better readability. Responsive design often uses relative units like em, rem, %, and vh/vw instead of fixed pixels.

Design Choices in This Portfolio

This site uses a minimal black-and-white theme (≈80% white background with black accents) for clarity and readability. Bootstrap utilities help with spacing, typography, and responsive breakpoints, while custom CSS refines the layout and theme. The goal is a clean, modern look that works across all devices.
