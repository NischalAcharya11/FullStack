Bootstrap — Using the Framework Effectively

Bootstrap is one of the most popular front-end frameworks. It provides a responsive grid system, prebuilt components (like navbars, cards, modals), and utility classes for quick styling. Instead of writing everything from scratch, developers can combine Bootstrap’s building blocks with custom CSS for both speed and consistency.

Responsive Grid Example

The grid system is Bootstrap’s core feature. It uses a .container → .row → .col-* structure, with breakpoints for mobile-first responsive design.

<div class="container">
  <div class="row g-4">
    <div class="col-md-6 col-lg-4">Column A</div>
    <div class="col-md-6 col-lg-4">Column B</div>
    <div class="col-md-6 col-lg-4">Column C</div>
  </div>
</div>

Here, three columns adjust automatically:

  • col-md-6 → two columns per row on medium screens.
  • col-lg-4 → three equal columns on large screens.
  • g-4 → spacing (gutter) between columns.

Customized Card Component

Bootstrap cards are versatile containers for content. Below is a card customized with a dark theme and a light button:

<div class="card card-invert">
  <div class="card-body">
    <h5 class="card-title">Black Card</h5>
    <p class="card-text">White text on a black background.</p>
    <a href="#" class="btn btn-light">Action</a>
  </div>
</div>

Cards can hold images, text, buttons, or even lists. By combining Bootstrap’s base styles with a custom class (.card-invert), this portfolio applies a black-and-white aesthetic consistently across pages.

Utilities for Speed

Bootstrap comes with hundreds of utility classes that reduce the need for custom CSS. For example:

  • p-* and m-* → padding and margin utilities (e.g., p-3 adds uniform padding).
  • border, rounded, shadow → quick borders and effects.
  • text-center, fw-bold, text-muted → typography helpers.

In this project, these utilities were used to make the layout responsive and visually consistent without writing large amounts of custom CSS.

Design Choices in This Portfolio

Bootstrap is used lightly here — mainly for the responsive grid, the navbar, and card structures. The rest of the design relies on a simple black-and-white theme, achieved through custom CSS. This balance ensures the portfolio feels personal while still benefiting from Bootstrap’s efficiency.
