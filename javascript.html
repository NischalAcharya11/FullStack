JavaScript — Fundamentals & Project

JavaScript is the programming language of the web. It makes static HTML and CSS interactive by enabling user events, dynamic DOM updates, and communication with APIs or storage. This page highlights key concepts—variables, functions, events, DOM manipulation, and local storage—and demonstrates them in a working To-Do List app.

Language Basics
// Variables & scope
let x = 1;        // block-scoped
const PI = 3.14;  // constant value

// Functions & events
function greet(name) { 
  return `Hello, ${name}`; 
}
document.addEventListener('click', () => 
  console.log('clicked')
);

// DOM manipulation
const el = document.querySelector('#app');
el.textContent = 'Updated by JS';

JS handles variables with "let" and "const", defines reusable functions, listens for events, and manipulates the DOM to update content dynamically.

Storage & JSON
// Save to localStorage
const data = { done: false, text: 'Task' };
localStorage.setItem('todos', JSON.stringify([data]));

// Load back
const items = JSON.parse(localStorage.getItem('todos') || '[]');

The localStorage API lets data persist across page reloads. Data is stored as JSON strings and retrieved using JSON.parse().

To-Do List (Working Example)

This mini project demonstrates how JavaScript powers real applications:

  • Form events capture user input.
  • Objects represent tasks ({text, done}).
  • localStorage saves tasks persistently.
  • DOM updates render, filter, and remove items.
    How it works: When the form is submitted, JavaScript creates a new task object, saves it in localStorage, and re-renders the list. Each item has a checkbox to toggle completion and a delete button for removal. The filter menu updates the view, and a clear button wipes completed tasks.
